# Airbnb Full Stack Clone

A full-stack Airbnb clone built with the MERN stack.

## Tech Stack

- React
- Vite
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Project Structure

```
AIRBNBCLONE/
├── frontend/
└── backend/
```

## Installation

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm start
```
